# 🏴‍☠️ WANTED: Das Daily One Piece Quiz

Willkommen beim ultimativen täglichen Quiz für alle One Piece Fans! 🍖

Jeden Tag erwarten dich **5 neue Fragen** aus der Welt von Eiichiro Oda. Kämpfe dich vom kleinen Matrosen bis zum Piratenkönig hoch und steigere dein Kopfgeld!

## 🔗 [Hier spielen!](https://paschkaop.github.io/DailyOP/)

---

## ✨ Features

* **Tägliche Herausforderung:** Jeden Tag 5 Fragen für die gesamte Community (synchron für alle Spieler!).
* **Ansteigende Schwierigkeit:** 2x Leicht, 2x Mittel und 1x Schwere "Königsfrage".
* **Daily Streaks 🔥:** Sammle Punkte für aufeinanderfolgende Spieltage.
* **Joker-System:** Nutze den 50/50-Joker oder Nico Robins Wissen.
* **OPWiki Integration:** Lerne nach jeder Frage mehr über die Hintergründe.
* **Sound & Design:** Authentischer One Piece Look & Feel.

---

## 🛠️ Technische Details

Dieses Projekt wurde komplett mit **HTML5, CSS3 und Vanilla JavaScript** umgesetzt. Es benötigt keine Datenbank und läuft stabil über GitHub Pages.

* **Zufalls-Logik:** Ein datumsbasierter Seed sorgt dafür, dass alle Nutzer weltweit am selben Tag die gleichen Fragen erhalten.
* **Speicherung:** Spielstände und Streaks werden sicher im `localStorage` deines Browsers gespeichert.
